## Concepts to learn first
- Overlapping subproblems — same sub-problem is solved multiple times in naive recursion
- Optimal substructure — optimal solution is built from optimal solutions of subproblems
- Memoization (top-down): recursion + cache (HashMap or array)
- Tabulation (bottom-up): iterative, fill table from base cases upward
- State definition: what parameters uniquely define a subproblem?
