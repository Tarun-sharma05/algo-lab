## Concepts to learn first
- Linear search — O(n)
- Binary search (iterative) — O(log n), requires sorted input
- Binary search (recursive)
- "Binary search on answer" concept — apply binary search to the answer space, not just indices

## Problems

### Easy
- [ ] Linear search — return index or -1
- [ ] Binary search on a sorted array (iterative)
- [ ] Binary search (recursive version)
- [ ] Find first and last occurrence of an element in sorted array

### Medium
- [ ] Search in rotated sorted array
- [ ] Find peak element (element greater than its neighbours)
- [ ] Find square root using binary search (integer part only)
- [ ] Find element in nearly sorted array (element may be at i-1, i, or i+1)

### Hard / Advanced
