## Concepts to learn first
- LIFO — Last In First Out
- Core operations: push, pop, peek, isEmpty
- Array-backed implementation vs linked-list-backed
- Monotonic stack — when to use it

## Problems

### Easy
- [ ] Implement Stack using a plain array (from scratch)
- [ ] Implement Stack using ArrayList
- [ ] Implement Stack using LinkedList (custom Node)

### Medium
- [ ] Valid parentheses / balanced brackets
- [ ] Next greater element (for each element, find the next element greater than it)
- [ ] Evaluate a postfix expression
- [ ] Min Stack — getMin() in O(1)
- [ ] Sort a stack using recursion (no extra data structure)
- [ ] Reverse a string using a stack

### Hard / Advanced
- [ ] Largest rectangle in histogram (monotonic stack)
- [ ] Daily temperatures
