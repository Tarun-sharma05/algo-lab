## Concepts to learn first
- String immutability and the String pool
- StringBuilder vs String (when and why)
- Key methods: charAt, length, substring, toCharArray, split, trim, compareTo, equals, contains, replace
- `==` vs `.equals()` for strings
- Character operations: Character.isDigit, Character.isLetter, Character.toLowerCase
