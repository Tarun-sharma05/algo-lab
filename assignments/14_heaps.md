## Concepts to learn first
- Min heap vs max heap
- Heap property: parent <= children (min heap)
- Heapify up (after insert) and heapify down (after delete)
- Array representation: parent at i, left child at 2i+1, right child at 2i+2
- Java PriorityQueue: min heap by default; use `Collections.reverseOrder()` for max heap
